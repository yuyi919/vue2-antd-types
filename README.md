# Install / Import

```bash
$ npm install --save vue2-antd-types
```

```typescript
import { myFunction, myObject } from "vue2-antd-types";
```

Specific imports:

```typescript
import { myFunction } from "vue2-antd-types/myFunction";
import { myObject } from "vue2-antd-types/myObject";
```
